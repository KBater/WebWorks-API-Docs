batteryStatus

static void blackberry.events.addEventListener("batteryStatus", yourCallbackFunction : function)

Supported Platform(s)

 - BlackBerry 10

Description

Allows you to attach a callback function to the battery status event. The callback is triggered whenever the battery level changes, the device starts to receive a charge and when the device no longer receives a charge.



Parameter Type Description
yourCallbackFunction function(level : Number, isPlugged : Boolean) The callback function that will be invoked on the batterystatus event

level: The percentage of the battery charge (0-100).
isPlugged: A boolean that represents whether or not the device is receiving a charge.

Code Example(s)

<script type="text/javascript">

function callBackFunc(level, isPlugged) {
  alert("Battery Charge: " + level + "%");
}

blackberry.event.addEventListener("batterystatus", callBackFunc);

</script>
