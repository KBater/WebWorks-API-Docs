window.navigator

window.navigator ()


Supported Platform(s)

 - BlackBerry PlayBook 1.0+
 - BlackBerry 10
 - Ripple Emulator

Description

Returns the Navigator object.



Code Example(s)

function supportNavigator() {
  if(window.navigator) {
   alert("supported");
  } else {
   alert("unsupported");
  }
}
