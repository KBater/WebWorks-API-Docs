FileEntry.file (successCallBack : function, errorCallBack : function)


Supported Platform(s)


Description

returns the file object associated to the fileEntry



Parameter Type Description
successCallBack function(File : File) function to be executed when successful

File: The new File object
errorCallBack function(FileError : FileError) function to be executed when an error is encountered

FileError: Invoked with a FileError object
