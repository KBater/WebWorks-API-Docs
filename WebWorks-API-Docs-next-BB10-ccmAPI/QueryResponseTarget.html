Property Type Description Supported Platform(s)
icon String The path of the icon that may be used to represent the target.
key String An identifier of target that supports the invocation.
label String The label that may be used to represent the target.
splash String The path of the icon that may be used to represent the viewer while loading. This field is present if the type property is VIEWER and is otherwise omitted.
type String Indication of the type of the invocation target. Possible values are "APPLICATION", "VIEWER" or "SERVICE".
