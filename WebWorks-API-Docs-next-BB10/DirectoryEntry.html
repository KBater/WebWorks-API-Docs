copyTo

void copyTo(original : DirectoryEntry, newName : String, callback : successCallback, callback : errorCallback)


Supported Platform(s)

 - BlackBerry PlayBook 1.0+
 - BlackBerry 10
 - Ripple Emulator

Description

Copies the directory to a new location. When copying to a location where the name already exists, copyTo will attempt to overwrite it.



Parameter Type Description
original DirectoryEntry directory
newName String new name of directory
callback successCallback() function to execute when it is successful
callback errorCallback() function to execute when there is a failure
