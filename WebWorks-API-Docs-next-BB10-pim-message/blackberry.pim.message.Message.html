API Reference

Message

The Message object represents an email item in the device. It can be obtained by calling blackberry.pim.message.MessageService.create() or blackberry.pim.message.MessageService.find()

void addAttachment (name : String, mimeTyep : String, filePath : String)


Add an attachment to the email.


Supported Platforms
 - BlackBerry 10


Parameters
name The display name that will be used for the attachment.
mimeTyep The mime type of the attachment that will be added.
filePath The file path of the attachment that will be added.

Code Example: 
function test() {
    var email = blackberry.pim.message.MessageService.create(),
        MessageAddress = blackberry.pim.message.MessageAddress;

    console.log("email.status: " + email.status); // status is "new" now

    email.addresses = [new MessageAddress(MessageAddress.TO, "Eric", "eric@rim.com"),
                       new MessageAddress(MessageAddress.TO, "Sergey", "sergey@rim.com"),
                       new MessageAddress(MessageAddress.CC, "Hassan", "hassan@rim.com"),];
    email.subject = "Test";
    email.body = "Some body";
    email.addAttachment("attachment1", "text/plain", "myTextAttachment.txt");
    email.send(onSuccess, onError);
}

void save (onSaveSuccess : function, onSaveError : function)


Save the message to draft or update the message with the changes. An error callback is called with an error code if an error happens.


Supported Platforms
 - BlackBerry 10


Parameters
onSaveSuccess The callback function that will be invoked when the message is saved successfully.
onSaveError The callback function that will be invoked when an error happens.

error: The detailed error code when an error happens.

void send (onSendSuccess : function, onSendError : function)


Send the unsent message. An error callback is called with an error code if an error happens.


Supported Platforms

Parameters
onSendSuccess The callback function that will be invoked when the message is sent successfully.
onSendError The callback function that will be invoked when an error happens.

error: The detailed error code when an error happens.
